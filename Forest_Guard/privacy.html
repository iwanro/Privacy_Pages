2. Ce Date Colectăm

Aplicația Forest Guard colectează următoarele date cu caracter personal:

  • Identificator dispozitiv (ANDROID_ID) — pentru autentificare și prevenirea utilizării abuzive
  • Token FCM (Firebase Cloud Messaging) — pentru trimiterea notificărilor push
  • Numere de înmatriculare introduse de utilizator — pentru monitorizare
  • Preferințe utilizator (temă, setări de monitorizare) — stocate local pe dispozitiv

Nu colectăm: nume, adrese de email (decât la contact direct), numere de telefon, poze, locație GPS reală, contacte, sau orice alte date personale sensibile.

3. Cum Folosim Datele

Datele colectate sunt utilizate exclusiv pentru:

  • Funcționarea Aplicației (monitorizare, notificări)
  • Îmbunătățirea serviciului
  • Prevenirea abuzurilor și a utilizării neautorizate

Datele nu sunt utilizate pentru publicitate, profilare, vânzare către terți sau orice alt scop comercial.

4. Stocarea Datelor

Datele utilizatorului sunt stocate în:

  • Dispozitivul local (Isar DB + SharedPreferences) — majoritatea datelor
  • Cloudflare Workers KV (server) — doar token FCM, numere de înmatriculare, ultimul ID de aviz procesat și tier-ul de abonament
  • Firebase — token FCM și date de diagnostic (Crashlytics, opțional)

5. GDPR — Drepturile Dumneavoastră

În conformitate cu Regulamentul General privind Protecția Datelor (GDPR), aveți următoarele drepturi:

  • Dreptul de acces — puteți solicita o copie a datelor stocate despre dumneavoastră
  • Dreptul la ștergere ("dreptul de a fi uitat") — puteți solicita ștergerea tuturor datelor dumneavoastră
  • Dreptul la rectificare — puteți corecta datele incorecte
  • Dreptul la portabilitate — puteți primi datele într-un format structurat

Pentru exercitarea acestor drepturi, contactați-ne la: iulianivn@duck.com

6. Parteneri Terți

Aplicația utilizează următorii furnizori terți:

  • Cloudflare — server și stocare KV (politica lor: cloudflare.com/privacypolicy)
  • Firebase (Google) — notificări push și raportare erori (politica lor: policies.google.com/privacy)
  • inspectorulpadurii.ro — sursa datelor publice (administrat de autoritățile române)

Furnizorul nu vinde, nu închiriază și nu partajează date personale cu terți, cu excepția celor necesari funcționării tehnice a Aplicației.

7. Securitatea Datelor

Implementăm măsuri tehnice și organizatorice rezonabile pentru protejarea datelor împotriva accesului neautorizat, pierderii sau modificării. Totuși, niciun sistem nu este complet sigur, iar Furnizorul nu poate garanta securitatea absolută a datelor.

8. Păstrarea Datelor

Datele sunt păstrate pe durata utilizării Aplicației. La ștergerea contului sau dezinstalarea Aplicației, datele sunt eliminate în termen de 30 de zile de pe server. Datele de pe dispozitivul local rămân până la dezinstalare sau ștergere manuală.

9. Modificări ale Politicii

Ne rezervăm dreptul de a actualiza această politică. Modificările vor fi publicate pe această pagină. Utilizarea continuă a Aplicației după modificări constituie acceptarea politicii actualizate.

2. What Data We Collect

The Forest Guard application collects the following personal data:

  • Device identifier (ANDROID_ID) — for authentication and abuse prevention
  • FCM Token (Firebase Cloud Messaging) — for push notification delivery
  • License plate numbers entered by the user — for monitoring
  • User preferences (theme, monitoring settings) — stored locally on device

We do NOT collect: names, email addresses (unless contacted directly), phone numbers, photos, real GPS location, contacts, or any other sensitive personal data.

3. How We Use Data

Collected data is used exclusively for:

  • Application functionality (monitoring, notifications)
  • Service improvement
  • Prevention of abuse and unauthorized use

Data is NOT used for advertising, profiling, sale to third parties, or any other commercial purpose.

4. Data Storage

User data is stored in:

  • Local device (Isar DB + SharedPreferences) — most data
  • Cloudflare Workers KV (server) — FCM token, license plates, last processed aviz ID, and subscription tier only
  • Firebase — FCM token and diagnostic data (Crashlytics, optional)

5. GDPR — Your Rights

Under the General Data Protection Regulation (GDPR), you have the following rights:

  • Right of access — request a copy of your stored data
  • Right to erasure ("right to be forgotten") — request deletion of all your data
  • Right to rectification — correct inaccurate data
  • Right to portability — receive data in a structured format

To exercise these rights, contact us at: iulianivn@duck.com

6. Third-Party Partners

The Application uses the following third-party providers:

  • Cloudflare — server and KV storage (their policy: cloudflare.com/privacypolicy)
  • Firebase (Google) — push notifications and error reporting (their policy: policies.google.com/privacy)
  • inspectorulpadurii.ro — public data source (operated by Romanian authorities)

The Provider does not sell, rent, or share personal data with third parties except as necessary for the technical operation of the Application.

7. Data Security

We implement reasonable technical and organizational measures to protect data against unauthorized access, loss, or alteration. However, no system is completely secure, and the Provider cannot guarantee absolute data security.

8. Data Retention

Data is retained for the duration of Application use. Upon account deletion or Application uninstallation, server data is removed within 30 days. Local device data remains until uninstallation or manual deletion.
